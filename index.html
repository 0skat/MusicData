Project Info

The purpose of this project is to compare accousticness,danceabilty, popularity, energy and tempo of eight songs. The songs chosen, are by some of my currently most listened to artist. This was determined by accesing my own user data through Spotify web services. A third-party dataset was used to compile the data for the selected songs and to create the graphs shown.

How To

On the left you can scroll through the data created with the songs in the playlist provided below. By clicking on the album cover, you can access that particular songs' data. Along with the data provided in these tables, each song has other data not provided in the graphs.
