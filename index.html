Welcome

Welcome to Music Data, a project that compares a variety of musical attributes between songs. For the scope of this project, only eight songs were selected based on the my likes and the Spotify data set that was acquired, through a third party source. Spotify measures acousticness, danceability, energy, instrumentalness, liveness, loudness, speechiness, tempo, and valence.

Click on any album to get the full stats

The values from the attributes mentioned above were used to draw comparisons between the selected tracks, which you can check out in the playlist above.


To view these graphical comparisons:

i Select the attribute from the drop down options
ii Populate and delete any attribute graph
iii Print or dowload a PDF of these graphs!
iv Enjoy!
