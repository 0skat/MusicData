Welcome to Music Data, a project that compares a variety of musical attributes between songs. For the scope of this project, I decided to curate eight songs that I love and had access to from a data set created by Spotify. Spotify measures acousticness, info regarding this danceability, energy, instrumentalness, liveness, loudness, speechiness, tempo, loudness, and valence. I took all these attributes and used their values to draw comparisons between the selected tracks, which you can check out in the playlist on your left. To view these graphical comparisons, simply populate the desired musical attribute that interests you. You can populate as many graphs as you wish and delete the ones you no longer want to display. If you wish to save or print any of these graphs use the print button.
